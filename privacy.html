Privacy

Effective May 8, 2026. Current build: AgentMash v0.3.0.

AgentMash is designed for private, local-first judgement of AI-generated work, with an optional API connection for teams that run the included server.

Data collection

The app does not collect, sell, or share personal data by default. Reviews, notes, uploads, and profile settings stay in browser storage on the device that runs the app unless the user connects an AgentMash API server and chooses to pull or send data.

User content

Images and text added to the deck are stored locally unless the user exports a JSON profile file or sends feedback to an AgentMash API server. Text profile data is stored in localStorage, and uploaded image bytes are stored in IndexedDB. Exported files are controlled by the user.

Deletion and retention

Local review data stays on the device until the user deletes it. The Reset profile control clears local profile data and uploaded image bytes. Users can also clear browser storage or uninstall the app to remove local data from that device. API-submitted artifacts can be deleted through DELETE /v1/artifacts/{artifactId} on the server.

Network use

The app does not call analytics, ads, AI APIs, or third-party services. The service worker only caches local app files for offline use. API calls happen only when the user configures the API connection and uses the sync controls.

Agent feedback packets

Feedback packets are generated in the browser from local review data. They can be copied, downloaded, or sent to a configured AgentMash API server.

Contact

Support details live on the support page. AgentMash does not send diagnostics automatically.
