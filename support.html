Support

AgentMash keeps reviews, notes, images, and feedback packets in the browser unless you export them.

Current build: AgentMash v0.2.0.

Before Asking For Help

Use Export to save a profile backup before importing another profile, clearing browser data, switching devices, or testing a new build.

Common Fixes

If the deck looks stale, refresh the page once. If a profile fails to import, confirm the file came from AgentMash export and has not been edited by another tool.

For image uploads, use PNG, JPG, or WebP files under 2.5 MB so the local browser profile stays reliable.

Getting Help

The app does not send diagnostics automatically. Keep an exported profile backup and the details below ready when troubleshooting a release.

What To Include

Include the browser, device, build version, the action you tried, and whether the issue affected local storage, import, export, or feedback packets.
