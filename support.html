Support

AgentMash keeps reviews, notes, images, and feedback packets in the browser unless you export them or use an API sync action.

Current build: AgentMash v0.3.0.

Back Up Your Profile

Use Export before importing another profile, clearing browser data, switching devices, trying a new build, or resetting the app. The exported profile is the portable backup.

Storage

Text profile data is stored in localStorage. Uploaded image bytes are stored in IndexedDB so the browser profile can handle larger decks more reliably.

If the app shows a Local storage full message, export a backup, reset the profile, or remove unneeded browser site data before continuing.

Import And Export

Import replaces the current local profile in this browser. Exported AgentMash profile files include review data and uploaded image bytes so they can be restored on another browser that supports IndexedDB.

API Connection

The Export workspace can pull queued artifacts from a same-origin AgentMash API server and send reviewed feedback bundles back. Use the token configured for that server. Leave the API URL blank when the app is served by the same server.

Common Fixes

If the deck looks stale, refresh the page once. If a profile fails to import, confirm the file came from AgentMash export and has not been edited by another tool.

For image uploads, use PNG, JPG, or WebP files under 2.5 MB so the local browser profile stays reliable.

Delete Local Data

Reset profile clears local review data and uploaded image bytes for this browser. Clearing browser site data or uninstalling the PWA also removes local AgentMash data from that device.

Server-side API artifacts are deleted with the authenticated deletion endpoint on the API server.

Diagnostics

The app does not send diagnostics automatically. Keep an exported profile backup and the details below ready when troubleshooting a release.

What To Include

Include the browser, device, build version, the action you tried, and whether the issue affected local storage, import, export, API sync, or feedback packets.
