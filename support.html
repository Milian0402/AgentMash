Support

AgentMash is a local-first beta candidate. Reviews, notes, images, and feedback packets stay in the browser unless you export them.

Current build: AgentMash v0.2.0.

Before Asking For Help

Use Export to save a profile backup before clearing browser data, switching devices, or testing a new build.

Common Fixes

If the deck looks stale, refresh the page once. If a profile fails to import, confirm the file came from AgentMash export and has not been edited by another tool.

Public Launch Contact

No public support inbox has been configured yet. Add a real support address or issue tracker here before publishing AgentMash to a public audience.

What To Include

When support is enabled, include the browser, device, build version, the action you tried, and whether the issue affected local storage, import, export, or feedback packets.
