# Store Listing Draft

## Name

AgentMash

## Subtitle

Human taste signals for AI work

## Short Description

Judge AI-generated websites, logos, copy, and product images with fast human intuition.

## Full Description

AgentMash is a local-first review deck for deciding whether AI-generated work is actually good enough to keep.

Swipe right when an output feels nice. Swipe left when it does not. Add a gut score, coherence score, craft score, usefulness score, tags, and notes. The app turns those human calls into structured feedback packets and JSONL eval rows.

Built for:

- Website screenshots and landing page concepts.
- Logo marks and brand directions.
- Generated copy and launch text.
- Product renders and image concepts.

Privacy:

- No accounts.
- No analytics.
- No AI API calls.
- No third-party tracking.
- Data stays on device unless exported by the user.
